Block "Banner Video"

This block includes video and text, it is designed like a banner and the text content can change the display position (left, right or center) with optional text color.

The button part (4) when clicking on the pen icon will appear a popup, you edit the link url (4a) and the link text (4b) of the button accordingly. You can use the search box (4c) to search for the page you want to navigate to. Click on the title (4d) you want to link to the cell URL (4a) that will automatically be updated again. Finally click "update" button (4)

Block "New product"

This block works like a slide, only the title (1) of the block can be changed and the content is displayed by default according to the latest product.

Block "Featured product"

This block defaults to displaying 4 products, only the block title (1) can be changed, and by default the content is displayed according to featured products (starred products in the admin section). The “Link view” (2) field selects the link you want to navigate when clicking the “view” button outside the interface

Block "Cate product"

This block defaults to displaying 3 products according to the selected category (1a), only the block title (1) can be changed. The “Link view” (2) field selects the link you want to navigate when clicking the “view” button outside the interface

Block "Interior"

This block has 2 columns, one column displays images with parallax effect, one column displays text content. If you want to change the left/right position of these two columns, change the "select position" field. Outside of the interface, to have a light gray background appear above the title, select the “Gray background” field. When this field is not selected, the default background color displayed is white

Block "Floricultura"

This block has 2 columns, each column has an image and content but the image size and text position are different. If you want to change the left/right position of these two columns, change the "select position" field. Outside the interface, to have a light gray background appear that takes up almost half of the block, select the “Gray background” field. When this field is not selected, the default background color displayed is white

Block "Flower Shop"

This block works like a slide, with 2 cell display options. 1 is image + title, 2 just shows text with default background color

Block "Project"

This block includes 2 separate parts: - Part of the image + text acts as a banner, - Partial works as photo slide. When clicking on the image, a small Popup will appear. You can zoom in on the image when you hover over the image

Add an image: click (2a) and a notification panel will appear -> then select the image you want to add, fill in the image information -> press the "Select" (2b) button.

Block "Banner Collection"

This block includes image and text, it is designed like a banner and the text content can change the display position (left, right or center) with optional text color.

Block "Paper Wrap"

This block includes images with somewhat unusually designed content. When clicking on the plus icon outside the interface, a collection window (4a) will appear (this collection will be clearly mentioned in the "Theme Settings" section).

Block "About Us"

This block also includes images and content, with a standalone title that remains stationary as the screen slides down. You can change the position of these two columns (left/right) in the "Select position" field.

Block "Banner About"

This block includes image and text, it is designed like a banner and the text content can change the display position (left, right or center) with optional text color

Block "Banner Short"

This block is designed like a banner with a shorter image that matches the concise title and text color that can be changed freely.

Block "Blog Category"

This block displays a short piece of text with a darker font color, suitable for the purpose of highlighting certain content.

Block "Blue White"

This block consists of 2 columns - a column displaying images - Column displaying vertical title content with color change effect (can choose color as desired)

Block "Description Single"

This block is designed to be simply a text block with an image

Block "Summy Single"

This block has 2 columns, one column displays images with parallax effect, one column displays image and text content. If you want to change the left/right position of these two columns, change the "select position" (1) field.

Install required plugins

Mkcafe themes use plugins Advanced Custom Fields Pro and Gravity Forms. Please install and active Advanced Custom Fields Pro and Gravity Forms.
On left menu, select “Appearance” -> Install Plugins
You will see list plugins as image blow.

Install required plugins

Install and active plugin “All-in-One WP Migration”.
After install that plugin, you can import example data into your site. You can see example data at folder lib/data/example-data.wpress themes folder.
On left menu, select “All-in-One WP Migrate” -> Import, and then select file example-data.wpress
Note: After import it will change user account to admin (password: admin)

Server Requirements

Server requirements involve several elements: RAM, database version, processing power, disk space, and the web server used. Here’s a quick look into the standard recommendations for a server when running WordPress:

  • Web server: Nginx or Apache with mod-rewrite module
  • Disk space: At least 1 GB
  • PHP: Version 7.1 or higher
  • Database: MySQL 5.015 or higher (An alternative is MariaDB, version 10.1 or higher)
  • RAM (Random Access Memory): At least 512 MB
  • CPU (Central Processing Unit): At least 1.0 GHz
  • Support for HTTPS
